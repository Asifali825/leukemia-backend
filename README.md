#  AI Leukemia Detection System

An AI-powered web application that detects leukemia from blood cell images using Deep Learning.

---

##  Features

- 🔐 User Authentication (Login/Signup)
- 🧪 Image Upload & Analysis
- 🧠 AI Model Prediction (Leukemia Detection)
- 📊 Confidence Score Display
- 🌐 Fully Deployed Web App

---

## 🛠️ Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python (Flask)
- TensorFlow / Keras

### Deployment
- Vercel (Frontend)
- Render (Backend)

## ⚙️ How It Works

1. User logs in
2. Uploads blood cell image
3. Image sent to Flask API
4. AI model predicts leukemia
5. Result + confidence displayed
